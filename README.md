## 支持什么功能?
  ```
  0.改阳光
  1.改金币
  2.免费植物
  3.黄油糊脸
  4.只出梯子僵尸
  5.只出巨人僵尸
  6.搭梯
  7.炸荷叶烂南瓜
  8.阵型压力测试
  9.过关
  10.修改当前无尽轮数
  11.跳关
  12.冒险跳关
  13.过第二周目
  14.混乱存档
  15.修改第一个卡槽
  16.切换场景
  17.自动收集
  18.取消自动收集
  19.输出调试信息
  (另,此版本有五个无尽(购买了第一个游戏礼包后可以从Mini Games里进入))
  ```
