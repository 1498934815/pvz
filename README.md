## 支持什么功能?
```
    1.改阳光
    2.免费植物
    3.黄油糊脸
    4.搭梯
    5.炸荷叶烂南瓜
    6.输出调试信息
    7.过关
    8.修改当前无尽轮数
    9.只出梯子僵尸
    10.阵型压力测试
    11.跳关
    12.混乱存档
    13.冒险跳关
    14.改金币
    15.修改第一个卡槽

```
