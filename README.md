## 支持什么功能?
  ```
  0.改阳光
  1.改金币
  2.免费植物
  3.关闭免费植物
  4.黄油糊脸
  5.只出梯子僵尸
  6.只出巨人僵尸
  7.搭梯
  8.炸荷叶烂南瓜
  9.阵型压力测试
  10.过关
  11.修改当前无尽轮数
  12.修改当前游戏模式
  13.冒险跳关
  14.过第二周目
  15.混乱存档
  16.修改第一个卡槽
  17.切换游戏场景类型
  18.自动收集物品
  19.取消自动收集
  20.触发场上推车
  21.输出调试信息
  (另,此版本有五个无尽(购买了第一个游戏礼包后可以从Mini Games里进入))
  ```
