## 支持什么功能?
```
    1.改阳光
    2.僵尸秒杀
    3.卷心菜投手加强
    4.免费植物
    5.黄油糊脸
    6.僵尸加强二倍
    7.植物血量增加二倍
    8.植物攻速极限
    9.搭梯
    10.炸荷叶烂南瓜
    11.自动发炮
    12.不自动发炮
    13.输出调试信息
    14.过关
    15.修改当前无尽轮数
    16.只出梯子僵尸
    17.阵型压力测试
    18.退出
```
